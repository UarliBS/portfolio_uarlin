# Portfolio Warllen Barreiros

Portfolio em React com Next.js, tema dark fixo, animações suaves e suporte PT/EN.

## Rodar localmente

```bash
npm run dev
```

Abra `http://localhost:3000`.

## Build

```bash
npm run build
```

## Onde editar

- `src/App.tsx`: textos em PT/EN, projetos, skills, links e seções.
- `src/styles.css`: visual, responsividade e animações.
- `src/app/layout.tsx`: metadados do site.
